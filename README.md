
# toGif

Simple bash script to convert video to gif.



## How to setup

Cloning the project

```bash
  git clone https://github.com/cand1z/toGif.git
```

Enter the project directory

```bash
  cd toGif
```

Giving execute permission to the script

```bash
  chmod +x toGif.sh
```



## Example of use


```console
 ./toGif.sh myfile.webm 
```
